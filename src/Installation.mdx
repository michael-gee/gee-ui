# Installation

## Prerequisites

Similar to [shadcn/ui](https://ui.shadcn.com/), this library is built on top of [react](https://react.dev/) and [tailwindcss](https://tailwindcss.com/) so you need to have them installed in your project.

- [React Installation Guide](https://react.dev/learn/installation)
- [Tailwindcss Installation Guide](https://tailwindcss.com/docs/installation)

### Install the package into your project

```bash
npm install @gee-ui/react
```

### If you plan to use `<Form />` or related components, install react-hook-form into your project:

```bash
npm install react-hook-form
```

## Add global styles

The following styles are the default values used within [shadcn/ui](https://ui.shadcn.com/docs/theming). You can override these styles to match your project's design system and if you don't want to override the default styles, you can just as easily override any styles at the component level.

## Add tailwind config

```ts
// tailwind.config.ts or tailwind.config.js
module.exports = {
  content: ['./src/**/*.{js,ts,jsx,tsx}', './node_modules/@gee-ui/react/dist/**/*.js']
};
```

```css
/* index.css or global.css */
@import 'tailwindcss';
@custom-variant dark (&:where(.dark, .dark *));
/* Add one of the 2 lines to the existing css file we modified above (wherever your tailwindcss import is) */
/* JavaScript file */
@config "path/to/tailwind.config.js";

/* OR */

/* TypeScript file */
@config "path/to/tailwind.config.ts";
```
