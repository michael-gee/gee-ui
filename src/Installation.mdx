# Installation

## Prerequisites

Similar to [shadcn/ui](https://ui.shadcn.com/), this library is built on top of [react](https://react.dev/) and [tailwindcss](https://tailwindcss.com/) so you need to have them installed in your project.

- [React Installation Guide](https://react.dev/learn/installation)
- [Tailwindcss Installation Guide](https://tailwindcss.com/docs/installation)

## Install the package into your project

```bash
npm install @gee-ui/react
```

## Add Tailwind Configuration

[Tailwindcss Configuration](https://tailwindcss.com/docs/configuration)

Add the following to your `tailwind.config.(js/ts)` file:

```js

import type { Config } from 'tailwindcss';

const config: Config = {
  // ...

  // Required additions:
  content: [(keep what already existed here) ..., './node_modules/@gee-ui/react/dist/**/*.{js,cjs}'],
  presets: [require('@gee-ui/react/tailwind-preset')],

  // Optional additions (dark mode support):
  darkMode: ['selector'], // (Use 'class' strategy if Tailwind CSS version is not yet v3.4.1 or higher)

  // ...

};

export default config;
```

## Add global styles

The following styles are the default values used within [shadcn/ui](https://ui.shadcn.com/docs/theming). You can override these styles to match your project's design system and if you don't want to override the default styles, you can just as easily override any styles at the component level.

Add the following to your `index.css` or `global.css` file:

```css
/* tailwindcss styles (these should already be here) */
@tailwind base;
@tailwind components;
@tailwind utilities;

/* gee-ui global styles */
@layer base {
  :root {
    --background: hsl(0 0% 100%);
    --foreground: hsl(240 10% 3.9%);

    --card: hsl(0 0% 100%);
    --card-foreground: hsl(240 10% 3.9%);

    --popover: hsl(0 0% 100%);
    --popover-foreground: hsl(240 10% 3.9%);

    --primary: hsl(240 5.9% 10%);
    --primary-foreground: hsl(0 0% 98%);

    --secondary: hsl(240 4.8% 95.9%);
    --secondary-foreground: hsl(240 5.9% 10%);

    --muted: hsl(240 4.8% 95.9%);
    --muted-foreground: hsl(240 3.8% 46.1%);

    --accent: hsl(240 4.8% 95.9%);
    --accent-foreground: hsl(240 5.9% 10%);

    --destructive: hsl(0 72.22% 50.59%);
    --destructive-foreground: hsl(0 0% 98%);

    --border: hsl(240 5.9% 90%);
    --input: hsl(240 5.9% 90%);
    --ring: hsl(240 5% 64.9%);
    --radius: 0.5rem;
  }

  .dark {
    --background: hsl(240 10% 3.9%);
    --foreground: hsl(0 0% 98%);

    --card: hsl(240 10% 3.9%);
    --card-foreground: hsl(0 0% 98%);

    --popover: hsl(240 10% 3.9%);
    --popover-foreground: hsl(0 0% 98%);

    --primary: hsl(0 0% 98%);
    --primary-foreground: hsl(240 5.9% 10%);

    --secondary: hsl(240 3.7% 15.9%);
    --secondary-foreground: hsl(0 0% 98%);

    --muted: hsl(240 3.7% 15.9%);
    --muted-foreground: hsl(240 5% 64.9%);

    --accent: hsl(240 3.7% 15.9%);
    --accent-foreground: hsl(0 0% 98%);

    --destructive: hsl(0 62.8% 30.6%);
    --destructive-foreground: hsl(0 85.7% 97.3%);

    --border: hsl(240 3.7% 15.9%);
    --input: hsl(240 3.7% 15.9%);
    --ring: hsl(240 4.9% 83.9%);
    --radius: 0.5rem;
  }
}

@layer base {
  * {
    @apply border-border;
  }
  body {
    @apply bg-background text-foreground;
    font-feature-settings:
      'rlig' 1,
      'calt' 1;
  }
}
```
